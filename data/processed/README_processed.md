# Processed Temperature Data

This folder stores outputs generated by `preprocess_temperature_pipeline()`.

Expected generated files:

- `temperature_clean.csv`: cleaned daily time series with canonical `date` and `temperature`.
- `temperature_train_scaled.csv`: chronological train split with `temperature_scaled`.
- `temperature_val_scaled.csv`: chronological validation split with `temperature_scaled`.
- `temperature_test_scaled.csv`: chronological test split with `temperature_scaled`.
- `temperature_scaled.csv`: all scaled rows with a `split` column.
- `split_info.json`: split sizes and date ranges.

Normalization rule:

- Fit standardization parameters on train only.
- Apply the train mean and standard deviation to validation and test.
- Save scaler parameters to `models/scaler_params.json`.

Time-series rule:

- Do not shuffle.
- Preserve chronological order for train, validation, and test.
